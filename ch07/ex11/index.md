### 動的配列の push の平均時間計算量
各再配置で要素ののコピー回数は

$1$, $2$, $4$, $8$, ..., $2^{\log_2 n}$

再配置の際にコピーの総和は

$2^0$ + $2^1$ + $2^2$ + $2^3$ + ... + $2^{\log_2 n}$

なので、計算してまとめると

$2n-1$

平均時間計算量はこれをnで割れば求められるので、

$2-1/n$ → O(1)
