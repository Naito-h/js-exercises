# 問題11.8
正規表現の処理には予想以上に時間がかかる可能性がある。

例えば、利用者が正規表現を入力して何か検索を行うことが出来るサービスを開発したとする。<br>
リリース後、とある利用者によって `^(a|aa)+$` といった文字列が入力されたと考えよう。<br>
この正規表現が `"aaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaa!"` といった文字列にマッチするか調べようとするとどうなるだろうか。

---

バックトラックが発生し、時間がかかる

この文字列は末尾に`!`が含まれているため、正規表現にマッチしないが、すべての組み合わせを試そうとする。

例えば、`"aaa"`の文字列の場合でも
- `a + a + a`, 
- `a + aa`, 
- `aa + a`

の3パターンを試すため、問題の文字列ではより多くのパターンを試そうとして、処理に時間がかかる。
