# 問題 11.4

ch11/ex04/index.js の実装を完成させ型付き配列と通常の配列で行列の乗算の速度を比較してみなさい。<br>
また実行する前にどのような結果になるか予測しなさい。

---

## 予想
型付き配列と通常の配列では
- Float64Arrayの場合、速度はほとんど同じになる
- Uint8Arrayの場合、型付き配列の方が速くなる

## 結果
### Float64Arrayのとき
```
arrayMultiply: 1230.6229000017047
typedArrayMultiply: 1382.9453999996185
```
型付き配列の方が、通常の配列より速度が遅くなった
- 型付き配列の要素は、配列が作成されたときに0に初期化されるため、初期化で時間がかかった

### Uint8Arrayのとき
```
arrayMultiply: 1258.8889999985695
typedArrayMultiply: 1082.13560000062
```
型付き配列の方が、通常の配列より速度が速くなった

- ロールオーバーすることで、コンピュータのメモリの低レベルの動作と同じになるため、速度が速い
