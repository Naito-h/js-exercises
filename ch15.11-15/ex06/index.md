# localStorage と sessionStorage に保存されたデータの有効期限

| 操作 | localStorage | sessionStorage |
|------|--------------|----------------|
| リロード（F5） | ○ | ○ |
| ページ遷移 | ○ | ○ |
| 履歴から戻る/進む | ○ | ○ |
| 別タブで開く | ○（共有） | ×（独立） |
| タブを複製 | ○（共有） | ○（複製される） |
| ブラウザを閉じて開く | ○ | × |
| サーバーを再起動 | ○ | × |

- localStorageはブラウザを閉じてもデータが保持される
- sessionStorageはブラウザまたはタブを閉じるとデータが破棄される
